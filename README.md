# Study Vault 🎓

A collection of interactive, self-paced revision courses for mastering complex technical subjects.

## Courses

### 🧠 Explainable AI (XAI)
- 9 modules covering interpretability techniques
- Topics: Intrinsic models, PFI, PDP/ICE, SHAP, LIME, CAM, Anchors, Counterfactuals
- Interactive flashcards, quizzes, and code examples
- **Access**: [xai/](xai/)

### 🔬 CS Research Methods
- 10 modules on research methodology and design
- Topics: Research fundamentals, literature review, paradigms, frameworks, quantitative/qualitative methods, ethics
- Module quizzes, comprehensive exam, concept note builder
- **Access**: [cs-research/](cs-research/)

## Features

- 📚 Structured notes with concepts and examples
- 💡 Key quotes and insights
- 📋 Module-level quizzes
- 🎯 Comprehensive final exams
- 🔍 Full-text search
- 📥 Download study materials
- ✏️ Concept note builder (for CS Research)
- 📊 Progress tracking

## How to Use

1. Visit the [landing page](https://ai-study-vault.github.io/)
2. Select a course
3. Work through modules sequentially
4. Take quizzes to test understanding
5. Complete the final exam
